# Consultas SQL Avanzadas - Gestión de Alojamientos Turísticos

Repositorio creado para la actividad académica **Consultas SQL avanzadas**, enfocado en la manipulación, consulta y análisis de bases de datos relacionales.

## Motor de Base de Datos Utilizado
* **SGBD:** PostgreSQL (Versión 17 o superior)
* **Herramienta de gestión:** Compatible con psql, DBeaver o pgAdmin.

## Descripción del Trabajo Realizado
Se desarrolló e implementó un script SQL completo (`consultas_avanzadas.sql`) que contiene un total de **20 consultas avanzadas** estructuradas sobre la base de datos `acomodation_db`. Las consultas abarcan:
* Filtrado y ordenamiento avanzado de datos (`WHERE`, `ORDER BY`).
* Funciones de agregación (`COUNT`, `SUM`, `AVG`).
* Consultas multitabla mediante uniones internas y externas (`INNER JOIN`, `LEFT JOIN`).
* Subconsultas anidadas y filtros lógicos complejos.
* Modificación y control de registros (`UPDATE`, `DELETE`).

## Estructura del Repositorio
* `consultas_avanzadas.sql`: Archivo fuente con las 20 consultas debidamente numeradas y documentadas por comentarios.
* `capturas/`: Carpeta que contiene las capturas de pantalla de los resultados obtenidos tras la ejecución exitosa de las consultas en PostgreSQL.
